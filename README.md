# RecombiLab 3D

Aplicativo educacional para simulação conceitual de etapas relacionadas à tecnologia do DNA recombinante.

O RecombiLab 3D inclui montagem de plasmídeo, mapa do pBluescript, enzimas de restrição, DNA ligase, seleção azul-branco, visualização de estruturas e cálculo conceitual da eficiência de transformação.

## Tecnologias utilizadas

O projeto utiliza HTML5, CSS3 e JavaScript para a interface e as simulações interativas. A versão Android foi construída no Android Studio, com Java e WebView Android, permitindo executar o aplicativo diretamente no celular. Também há uma versão com backend em Python usando FastAPI e Uvicorn para execução local durante o desenvolvimento. A compilação do APK é feita com Gradle.

## APK

O arquivo `RecombiLab-3D-debug.apk` é uma versão Android com funcionamento offline, sem necessidade de manter o backend ligado.

## Disponibilização

Este repositório será disponibilizado no GitHub para acesso ao código-fonte, arquivos do aplicativo e materiais relacionados ao desenvolvimento, permitindo consulta, estudo e adaptação para fins educacionais.
